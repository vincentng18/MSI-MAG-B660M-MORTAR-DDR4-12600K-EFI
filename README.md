## BIOS设置

- 关闭CSM（启动模式仅UEFI）
- 关闭快速开机 Fast Boot
- 关闭MSI快速开机
- 关闭Intel VT-D
- 关闭CFG Lock
- 最好屏蔽核显
- 大小核均可开启
- 超线程可开启
